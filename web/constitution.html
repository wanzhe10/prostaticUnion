世界内镜医师协会章程

(2019年打印)

为造福人类大健康卫生，全球同行学者携手共建本学科 发展，加入本协会的各国同仁，为维护本人的学术荣誉、权利 和义务，履行本人的社会责任，依国际惯例，制定世界内镜医 师协会章程。

第一章    总则

[ 第一条 ]

本会名称为世界内镜医师协会，英文名称为World Endoscopy Doctors Association，缩写为WEDA。“世界内镜医师协会(WEDA)”成立于2009年， 在香港特别行政区登记注册，现总部位于美国洛杉矶。本协会2010年9月17日 经中华人民共和国科学技术部国科外(2010)0475号文件批准,世界内镜医师协会 作为国际卫生科技组织设立北京代表处，是新中国建国至今，批准的唯一一家 在中国建立“代表处(总部)”的医学科技世界联盟医师组织机构。

[ 第二条 ]

(一)世界内镜医师协会(以下简称本会)是由2009年中国卫生部肝胆肠外科研究 中心、恩德思医学科学技术奖评奖委员会、中国中南大学内镜医学研究院、中南大学 生物工程研究院、中国医师协会内镜医师分会、中国医药生物技术协会纳米生物技术分 会、美国内镜微创外科研究院、乌克兰敖德萨国立医科大学、《中国内镜新闻》、《健康 前沿》、《世界内镜医师协会网》等机构，与世界多国和地区(现已37国与地区)的医学 协会、学会组织，多专科内镜微创医师与内镜研制相关的理学、工学的科学技术研究人 员、工程技术研究人员自愿参加组成的，内镜医学科学技术国际联盟组织协会。中国籍 医学家、教育家、科学家张阳德教授，当选为协会主席、大会主席团主席。世界内镜医 师协会以下依各专科行业分别建立相应协会。

(二)本协会2010年9月17日经中华人民共和国科学技术部国科外批字(2010) 0475号文件批准，在中国北京设立常驻代表处。业务范围:“有关港澳台地区与 内地及世界各国内镜医师和学术机构在内镜行业医师交流及合作方面的联络工作。 北京代表处首席代表是张阳德教授。”

(三)本协会是一个为人民健康提供服务保障。学术性、社会公益性、非盈利性非 政府的世界内镜医师联盟组织。得到世界多国政府健康管理机构、专业学术团体、 医疗机构和科学技术研究机构、公司企业、社会团体、国际著名医学、理学、工学 科学家的支持关注。是发展世界内镜与微创医学科学技术的重要世界科学技术医师 联盟组织。

(四)关心、帮助基层医疗机构的医师，开展健康医疗扶持工作，造福各国医疗与 健康管理水平提高;搭建医师与政府、人民群众健康平台，促进医师与社会各界的 交流和联系，营造并构建和谐有序的医疗环境和医疗秩序。

[ 第三条 ]

本协会为医学、理学、工学多专科国际联盟的内镜与微创医师行业组织。将与世 界各国内镜医师共同探讨、建立统一规范的内镜医师培训考核机制。探讨各国对各专 科的内镜医师进行统一规范、系统的内镜理论学习和操作培训。探求在各国“统一规 范、国际认证内镜与微创医师”质量评价标准、持证上岗。推进大健康进步。

[ 第四条 ]

制定并严格实施行业管理标准，为人类健康事业服务。本会依法维护内镜医师、技师、内镜与微创研制科学技术人员的合法权益。本会积极联络世界各国大学、医疗 机构、科研院所、公司的医学、理学、工学科技人员，研制各类高技术的新型内镜及器 材、内镜机器人、内镜消毒设备等，并将纳米生物技术应用于内镜微创医学设备与器材，医 学、理学、工学联合，推进世界内镜发展。

[ 第五条 ]

本会北京代表处办公地址:中国北京市西直门外大街甲143号凯旋大厦C 座4层。

第二章    业务范围

[ 第六条 ]

本协会业务范围

（一）促进世界各国内镜医学发展。开展内镜下微创医学基础研究和临床应用研究， 提高临床应用水平。

（二）促进医学、工学、理学多学科交叉融合，积极研制、推广用于弯曲管道深部探 查狭窄缝隙远距离观察的新型内镜设备和器材。促进将内镜用于医学等多领域。

（三）积极开展生物医学工程学科在内镜领域的研究、教学、医学诊断与临床治疗 工作。

（四）促进纳米生物技术与内镜微创设备与器件研制，积极推广微小微型高新技术 内镜产品。

（五）从事医学保健与健康教育、内镜相关专业技术人员培训与继续教育。

（六）从事培训基地设计、策划。制定质量评价标准、机构运行管理措施、成本核算、 效益评估。

（七）从事医用设备、器材、药品研制、开发、推广。

（八）从事健康经济学、医学经济学研究。

（九）推进《中国内镜新闻》、《健康前沿》、《世界内镜医师协会网》出版发行。推 进《中国内镜新闻网》、《世界内镜医师网》多国语言全球发布。

第三章    会员

[ 第七条 ]

本会会员类别分为个人会员和机构团体会员两类。

[ 第八条 ]

（一）个人会员:中华人民共和国内地的会员需获得中级以上医技职称证、医师执 业证、医师资格证，从事内镜医技工作3年以上的医技人员。香港、澳门、台湾地区、 及其他国家的会员，应获得医师执照、医师证书。从事内镜医技工作三年以上医技人员。 或根据不同专科专业的要求，经过专科医师培训，已取得专科医师资格者。世界各国 的理学、工学及其它学科从事与内镜领域相关研究的光电子、材料、纳米生物、药物、精 密机械加工、图像处理、信息处理、内镜研制、内镜消毒与灭菌设备、内镜消毒与灭菌 药品制剂健康、保健教育等，该行业中级技术职称以上的科技人员、管理人员。

（二）单位团队会员:与本协会专业有关、自愿参加本协会活动，支持本协会工作， 具有一定社会影响，有一定数量医学或内镜相关专业科技人员依法成立，并已经登记 的社团组织、医疗卫生、健康教育、科学研究机构、学校、事业、公司等机构。

[ 第九条 ]

入会程序:

（一）在本会网站下载入会表格，在线注册申请。

（二）提交入会申请书。

（三）由理事会或理事会授权的机构颁发会员证书。

[ 第十条 ]

（一）进入本会管辖范围的世界各国分会和培训基地网站。

（二）享用《健康前沿》、《世界内镜医师协会网》等电子数据库。

（三）享有每年《世界内镜医师大会》会议注册费及培训继续教育费用优惠。 员权益:

（四）优先安排内镜技术培训。

（五）享有申报办理中国银行和世界内镜医师协会联名卡发出的《长 城世界内镜医师 联名卡》即“行业高端身份荣誉卡”。和高额透支的美元、人民币贵宾信用卡。

[ 第十一条 ]

第五章    世界内镜医师大会

[ 第十二条 ]

世界内镜医师大会为年度一次的世界内镜医师年会。大会宗旨为为全球从事内 镜事业工作的组织与个人，提供来自世界各地各专科内镜专家进行学术交流的平台。展 现最新医学成果，提升本专业协会组织的国际影响力。首届世界内镜医师大会于2009年12 月18日-21日在香港亚洲国际博览馆举行，中国近300家内镜培训基地均派出专家和领导与会。中国香港、澳门、台湾、英国、法国、德国、澳大利亚、新加坡、乌克兰等国家和地区 1500多名代表参加了大会。

第六章    使用原则

[ 第十三条 ]

本会经费来源:

（一）按照有关规定收取会员会费。

（二）国(境)内外个人或大学、医院、科研院所、企业、团体的捐赠。

（三）大会会议收入和本协会基金。

（四）有关部门资助。

（五）广告费、咨询费、推广费、培训费、服务费。

（六）捐赠。

（七）本协会从事的评审费、出版费、稿费、质量评价标准设计费、产品研制费、科 研协作费、各培训考试、基地设计费、产品开发费、展示费。

（八）各种收入的利息。

（九）其它合法收入。

[ 第十四条 ]

本会经费收入，用于本会业务范围的活动和事业发展，不能将收入作为利益进行分配。

[ 第十五条 ]

本会经费管理:

（一）本会经费收支执行所在国家或地区有关科技社团非盈利性的财经法规和制度规定。

（二）本会建立严格的财务管理、财务审计制度，保障会计资料合法、真实、准确、完整。

（三）本会配备具有专业资格的专职会计人员。会计不兼任出纳。会 计人员要进行核 算，实行会计监督。会计人员调动工作或离职时，必须与接管人员办理交接手续。

（四）本会需委任合格的，专业的，严谨的核数师，依各国《非盈利非政府社会学术 组织》的规定，定期对资金进行财务审计。
